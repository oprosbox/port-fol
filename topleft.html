  • приложение для визуальной разработки торговых роботов(Windows) скачать

    • Приложение разработано в рамках ВКР и представляет собой самостоятельный инструмент разработки торговых роботов. С целью упрощения процесса разработки была создана библиотека индикаторов рынка и простых функций. Информация о структыре торгового робота хранится в виде XML файла. Инструмент разработки по визуальному представлению торгового робота в виде графа генерирует XML код. Затем торговый робот проходит через оптимизацию и на выходе получается набор адаптированных под заданный интервал времени торговых ботов. Они уже могут быть запущены на online - работу с рынком. Приложение многопоточное. С небольшими доработками может стать кросс платформенным. Для обработки XML файлов применяется технология DOM. Проект написан на Qt
